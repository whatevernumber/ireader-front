# iReader

A small site for a book store. Uses RESTful API from another project - [iReader](https://github.com/whatevernumber/ireader-test);
Site is available at http://89.191.225.149

# Stack

SvelteKit, Typescript, Tailwind, DaisyUI.

# Local development
```bash
npm install
npm run dev
```
# Building

```bash
npm run build
```
